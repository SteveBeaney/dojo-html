Web page to demonstrate pictures and movies.
Pictures: It's easy to include pictures using the <img>

To find the url of a picture on the WWW: find a picture, right click on it, select 'copy image address'. Now goto and editor new file and paste the link in.
Movies: It's easy to include you-tube movies using the <iframe>.
You need to get the text for the link from the you-tube site. Find a movie, goto share, select embed and copy out the html it shows you.
